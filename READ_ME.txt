AVIAGENCY v0.01

-------------------------------
Aviagency is tool that provides user a clean interface to book air tickets and save them in the database. 
Includes calculating price, availability of flights and seats, login, sign up, check booked tickets and cancel booked ticket. 
The front is carried out with Html and CSS, backend supports python and flask, and MySQL as the database.
-------------------------------

NOTE: The database used for this prototype is local MySQL database. ALl the flight details has been entered manually.
